# FilePicker
Android File Selection Library<br /><br />

## USAGE INSTRUCTIONS<br /><br/>
* Add jitpack repository to your root build.gradle file
```
allprojects 
{
	repositories 
	{	
		...
		maven { url 'https://jitpack.io' }
	}
}
```
* Add the dependency
```
dependencies 
{
	compile 'com.github.amogh93:FilePicker:2aa41268e6'
}
```
